Setting Up

  1. Make sure that the install path is c:\program files\LEMu2p0\ (so that the file LEMu2.bat is within this directory)
  2. Run the program c:\program files\LEMu2p0\msDrivers.exe
  3. You will see an array of boxes.
    1. Click on the top left hand box.
    2. On the left, click on the input devices you want to use in LEMu2, eg the MIDI device that your MIDI controller is plugged into.
    3. On the right, click on the output devices you want LEMu2 to send to. Eg, you might be using MIDIyoke to route MIDI to Reason, so you could select MIDIYoke bus 1 and select Reason to recieve MIDI on MIDIYoke bus 1.
  4. Close the msDrivers program. MIDI is now setup.
  5. run the LEMu2.bat file
