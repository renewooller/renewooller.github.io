Setting Up

  1. Make sure that the install path is c:\program files\LEMu2p0\ (so that the file LEMu2.bat is within this directory)
  2. Run the program c:\program files\LEMu2p0\msDrivers.exe
  3. You will see an array of boxes. Click on the top left hand box. On the left, click on the input devices you want to use in LEMu2, eg the MIDI device that your MIDI controller is plugged into. On the right, click on the output devices you want to
