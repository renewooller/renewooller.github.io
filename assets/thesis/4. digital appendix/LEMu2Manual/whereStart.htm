Where do I start?

If you have never used midishare before, you will need to goto 1.setting up MIDI. After that just go through the other items in the menu to your left to learn how to use the program.

If you are on a mac, you will need to install the mac version of midishare yourself. The program is currently untested on mac, but it should work "in theory" as much as Java is 'theoretically' a cross platform language.
